Tags

O que são as tags de formatação?

As tags de formatação em HTML são elementos utilizados para controlar a apresentação e o estilo do conteúdo em uma página da web. Elas permitem que os desenvolvedores especifiquem como o texto, imagens e outros elementos devem ser exibidos no navegador do usuário. As tags de formatação são uma parte essencial da criação de páginas da web bem projetadas e visualmente atraentes.
Saiba mais em developer.mozilla.org

Atributos de Eventos

Os eventos disparados pelas tags HTML podem disparar scripts no JavaScript da página se eles existirem. Contudo o script de tratamento de evento da tag HTML não é obrigatório, ou seja, o evento sempre ocorrerá mas se não houver uma rotina para tratamento deles não irá ocorrer qualquer erro. Definimos scripts para tratar apenas os eventos que nos interessam.
